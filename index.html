eclipse java formatter - google style

发表于 2017-10-15

谷歌代码规范文件：
https://raw.githubusercontent.com/google/styleguide/gh-pages/eclipse-java-google-style.xml
可以通过wget下载或者其他方法，复制都行。

导入方法：
Eclipse->window->preferences->Java->Code Style->Formatter->Import 选择eclipse-java-google-style.xml文件。
默认的缩进是2格，一般习惯都是四格的缩进，可以在edit里面修改一下。

阿里巴巴代码规范检测插件
addr:github 搜p3c
Eclipse版在线安装：

Help >> Install New Software
then enter this update site URL https://p3c.alibaba.com/plugin/eclipse/update
装好之后右键就能看见选项，直接用了。

MySQL安装

发表于 2017-10-15

1、mysql安装

安装mysql和mysql-devel都成功，但是安装mysql-server失败

2、下载安装mysql-server；

安装成功后重启mysql服务

附：
启动mysql

关闭mysql

查看运行状态

初次安装mysql，root账户没有密码

> mysql -u root

设置密码 

mysql> set password for 'root'@'localhost' =password('password');

3、配置mysql编码UTF-8
mysql配置文件为/etc/my.cnf 最后加上编码配置

[mysql]    
default-character-set=utf8
[client]
default-character-set=utf8
[mysqld]
character-set-server=utf8

4、远程连接设置
把在所有数据库的所有表的所有权限赋值给位于所有IP地址的root用户。 

mysql> grant all privileges on *.* to root@'%'identified by 'password';

如果是新用户而不是root，则要先新建用户

mysql> create user 'username'@'%' identified by 'password';

此时就可以进行远程连接了。

5、卸载

$ yum remove  mysql mysql-server mysql-libs mysql-server

将找到的相关东西delete掉 

$ find / -name mysql

查询出来的东东yum remove掉

$ rpm -qa|grep mysql
